Pocket Finance
O Pocket Finance é um gerenciador de despesas pessoais desenvolvido em Kotlin. O projeto utiliza o ecossistema Firebase para oferecer uma experiência de usuário fluida, com autenticação e sincronização em tempo real.

Funcionalidades
Autenticação: Sistema de login e cadastro seguro via Firebase Auth.

Realtime Database: Registro de despesas com atualização instantânea.

LocalStorage: Persistência de dados para consulta mesmo quando estiver offline.

UI Dinâmica: Interface adaptativa com suporte a Temas Claro e Escuro (Material Design 3).

Gestão: Adição rápida de gastos e exclusão via clique longo.

Tecnologias Utilizadas
Linguagem: Kotlin

Banco de Dados: Firebase Realtime Database

Autenticação: Firebase Authentication

Layout: XML com Material Design components

Recursos: Uso de strings.xml e colors.xml para fácil manutenção e acessibilidade.

Instalação e Teste
Clone o repositório:

Bash
git clone https://github.com/SEU_USUARIO/PocketFinance.git
Por motivos de segurança, o arquivo google-services.json (Firebase) foi omitido deste repositório.

Para rodar o projeto, crie um projeto no Firebase, gere o arquivo de configuração e coloque-o na pasta app/.
